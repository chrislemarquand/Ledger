LOGBOOK (ExifEditMac) - App Summary

WHAT IT IS
Logbook is a local macOS app for viewing and editing photo metadata using ExifTool.
It supports EXIF, IPTC, and XMP reads/writes with backup-first edits and restore.

WHO IT'S FOR
Primary persona: Not found in repo.
Inferred from code: photographers/photo editors who batch-manage image metadata and geotagging.

WHAT IT DOES
- Opens folders and browses files in gallery or list view with sort/search controls.
- Reads metadata via ExifTool JSON output and maps fields into EXIF/IPTC/XMP snapshots.
- Lets users stage metadata edits across selected files and apply in batch.
- Creates operation backups and restores files from backup by operation ID.
- Supports preset create/edit/save/apply with JSON persistence in Application Support.
- Imports GPX track points and stages GPS metadata updates with conflict resolution.
- Provides Quick Look preview, thumbnail generation, Finder reveal, and open-in-default-app actions.

HOW IT WORKS (REPO-EVIDENCE ARCHITECTURE)
- UI layer (`Sources/ExifEditMac`): `ExifEditMacApp`, `NativeThreePaneSplitViewController`, and SwiftUI views.
- State/orchestration (`AppModel`): selection, staged edits, undo/redo, progress, status, presets, import flows.
- Core engine (`Sources/ExifEditCore`): `ExifEditEngine` validates patches, makes backups, invokes ExifTool service.
- Service layer: `ExifToolService` builds/runs ExifTool commands and parses JSON into domain models.
- Data/storage: metadata domain structs in `Domain.swift`; backups under Application Support; presets in `presets.json`.
- Data flow: folder/files -> `readMetadata` -> `FileMetadataSnapshot` -> staged edits in `AppModel` -> `apply` -> backup + ExifTool writes -> `OperationResult`.

HOW TO RUN (MINIMAL)
- Dedicated quick-start doc: Not found in repo.
- Prereqs from repo: macOS 15+, Xcode CLI tools, and ExifTool 13.51 (see `Config/Base.xcconfig` and `scripts/build/bundle_exiftool.sh`).
- Build: `xcodebuild -project ExifEditMac.xcodeproj -scheme ExifEditMac -configuration Debug -destination 'platform=macOS' build`.
- Run: open `ExifEditMac.xcodeproj` in Xcode and run the `ExifEditMac` scheme.

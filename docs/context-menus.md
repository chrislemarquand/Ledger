# Ledger — Context Menu & Menu Bar Items

Edit this file and return it to update menu labels.
Lines starting with `#` are comments. Indented items are submenus.

---

## Sidebar (right-click on folder)

- Open in Finder
- Pin
- Unpin
- Move Up
- Move Down

---

## Browser — Gallery View (right-click on image)

- Open in [App]         ← dynamic: e.g. "Open in Preview"
- Reveal in Finder
- ---
- Apply Metadata Changes to [N] Image(s)  ← dynamic count
- Refresh Metadata
- Clear Metadata Changes
- Restore from Backup

---

## Browser — List View (right-click on row)

Same items as Gallery View context menu (above).

---

## Menu Bar — File Menu

- Open Folder…          (Cmd+N)
- ---
- Open                  (Cmd+O)
- Open With             → submenu: compatible apps, or "No Compatible Apps"
- Reveal in Finder
- Quick Look            (Cmd+Y)
- ---
- Pin Folder         (Cmd+Opt+T)
- Unpin Folder
- Move Folder Up
- Move Folder Down

---

## Menu Bar — Edit Menu

- Undo                  (Cmd+Z)
- Redo                  (Cmd+Shift+Z)
- ---
- Cut                   (Cmd+X)
- Copy                  (Cmd+C)
- Paste                 (Cmd+V)
- Select All            (Cmd+A)
- ---
- Rotate                (Cmd+R)
- Flip                  (Cmd+Shift+F)

---

## Menu Bar — View Menu

- as Gallery            (Cmd+1)
- as List               (Cmd+2)
- ---
- Sort By →
    - Name              (Cmd+Ctrl+Opt+1)
    - Date Created      (Cmd+Ctrl+Opt+2)
    - Size              (Cmd+Ctrl+Opt+3)
    - Kind              (Cmd+Ctrl+Opt+4)
- ---
- Zoom In               (Cmd++)
- Zoom Out              (Cmd+-)
- ---
- Toggle Sidebar        (Cmd+Opt+S)
- Toggle Inspector      (Cmd+Opt+I)

---

## Menu Bar — Image Menu

- Apply Metadata Changes to Selection       (Cmd+S)
- Refresh Metadata for Selection            (Cmd+Shift+R)
- Clear Metadata Changes                    (Cmd+K)
- Restore from Backup                       (Cmd+B)
- ---
- Apply Metadata Changes to All Images      (Cmd+Opt+Shift+S)
- Refresh Metadata for All Images           (Cmd+Opt+R)
- Clear Metadata Changes from All Images    (Cmd+Opt+K)
- Restore Metadata from Backup for All Images (Cmd+Opt+B)
- ---
- Apply Preset →
    - [preset names, or "No Presets"]
    - ---
    - Save as Preset…
    - Manage Presets…

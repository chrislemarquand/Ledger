-Context Menu Structure (Ledger)
--Scope rule
---Right-click on unselected item: selection becomes that single item, then menu actions target that item.
---Right-click on selected item: menu actions target the current multi-selection.

--Browser List Context Menu
---Open [dynamic title; system symbol/action state driven]
---Reveal in Finder
---[Divider]
---Apply Metadata Changes to Selection [dynamic count title]
---Refresh Metadata for Selection [dynamic title/state]
---Clear Metadata Changes [dynamic title/state]
---Restore from Backup [dynamic title/state]

--Browser Gallery Context Menu
---Open [dynamic title; system symbol/action state driven]
---Reveal in Finder
---[Divider]
---Apply Metadata Changes to Selection [dynamic count title]
---Refresh Metadata for Selection [dynamic title/state]
---Clear Metadata Changes [dynamic title/state]
---Restore from Backup [dynamic title/state]

--Notes
---List and Gallery context menus are currently the same command set and ordering.
---No keyboard shortcuts are assigned directly on context-menu items.
---Enabled/disabled state is computed from `fileActionState(...)` for the target URLs.
